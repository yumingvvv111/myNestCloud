# Notadd Rpc Demo

Notadd 微服务示例

## 使用说明

本仓库是微服务的 `Server` 端，单独可以运行，如需调用此 Rpc 服务，需要使用 `Notadd Application` 作为 `Client` 端来对外提供 GraphQL 接口

`Notadd Application` 仓库地址： [Notadd](https://github.com/notadd/notadd)

### 安装依赖

```bash
# yarn install
$ yarn install
```

### 启动

```bash
# 启动 Rpc 服务
$ yarn start
```
